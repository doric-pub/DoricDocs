常见问题

页面空白

优先检查：

  • src 地址是否能被浏览器直接访问。
  • Bundle 是否已经通过 doric build 生成。
  • 页面容器是否有明确宽高。
  • 浏览器控制台是否存在加载失败或跨域错误。

初始化数据读取不到

确认 data 属性是合法 JSON 字符串，并在 JS 侧通过 this.getInitData() 获取。

自定义协议加载失败

确认已经在 <doric-div> 开始加载前调用 registerDoricJSLoader，且 filter(source) 能正确匹配 src。
