命令

create

新建一个Doric项目，项目名为ProjectName。

dev

开始开发Doric页面，此时终端会输出二维码。可通过Doric Playground扫码查看页面效果

build

进行代码打包，此时输出JS bundle文件至 bundle 目录下。

clean

清理编译缓存文件。此时会删除build和bundle文件夹中内容
