---
title: Progressive Adoption
---

Doric can be adopted at app, page, embedded view, or web element level.

## Levels

- Full Doric app with `doric create`.
- Native page with `DoricActivity` or `DoricViewController`.
- Embedded region with `DoricPanel`.
- Web rendering with `<doric-div>`.

## Recommendation

Use Doric when you need dynamic cross-platform pages inside an existing host app.
