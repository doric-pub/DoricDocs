---
title: Doric Devkit
---

Doric Devkit provides logging, hot reload, and debugging support during development.

## Integration

- Add `DoricDevkit` only in debug builds.
- Open the Devkit panel from the host app.

## Usage

- Run `doric dev`.
- Scan the QR code or input the printed IP.
- Use hot reload and logs from the connected app.
